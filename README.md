# Library Management System

## Description
This is a PHP and MySQL based Library Management System project.
It has Admin and User roles with role-based access control.

## Technology Used
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL

## How to Run the Project
1. Install XAMPP
2. Start Apache and MySQL
3. Copy project folder into htdocs
4. Create database named `library`
5. Create tables using phpMyAdmin
6. Open browser and run:
   Project runs locally using XAMPP:
   http://localhost/Library-Management-System/
(http://localhost/Library-Management-System/admin.php)
